## jpegoptim

### 安装及使用

```shell
# 安装
brew install jpegoptim
# manual
man jpegoptim

# 在图床仓库中压缩以往占用空间较大的原图，限制在300k以内
jpegoptim --size=300k blog*.jpeg
```

## optipng

### 安装及使用

```shell
# 安装
brew install optipng
# manual
man optipng

# 使用
optipng IMG_0344.PNG
```
